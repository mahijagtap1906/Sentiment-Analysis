# Sentiment-Analysis
Developed a Sentiment Analysis model using Python, NLP, TF-IDF, and Logistic Regression. The project preprocesses text data and classifies sentiments as Positive, Negative, or Neutral with machine learning techniques.
# Sentiment Analysis Using Machine Learning

## Overview

This project performs Sentiment Analysis on text data using Natural Language Processing (NLP) and Machine Learning techniques. The model predicts whether a given text is Positive, Negative, or Neutral.

## Features

* Text preprocessing and cleaning
* TF-IDF Vectorization
* Logistic Regression Model
* Sentiment Prediction
* Model and Vectorizer saved using Joblib

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* NLTK
* Joblib

## Project Workflow

1. Data Collection
2. Text Preprocessing
3. Feature Extraction using TF-IDF
4. Model Training with Logistic Regression
5. Model Evaluation
6. Sentiment Prediction on New Text

## Files

* `Sentiment_analysis.ipynb` - Main Notebook
* `sentiment_model.pkl` - Trained Model
* `tfidf_vectorizer.pkl` - Saved TF-IDF Vectorizer

## Example

Input:
"I love this product"

Output:
Positive

Input:
"This service is terrible"

Output:
Negative
